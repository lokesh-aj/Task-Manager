# Task Manager API

Task Manager API is a Spring Boot REST API project built to understand real backend development concepts using Java, Spring Boot, Hibernate, and MySQL.

The project focuses on learning layered architecture and how different backend components interact with each other in a real application.

This project demonstrates:

- REST API Development
- Spring Boot Backend Architecture
- Controller Layer
- Service Layer
- Repository Layer
- DTO Architecture
- Hibernate & JPA
- MySQL Integration
- CRUD Operations
- Entity Relationships
- One-To-Many Mapping
- Many-To-One Mapping
- Dependency Injection
- ORM (Object Relational Mapping)

The application allows users to create and manage tasks while maintaining proper relationships between users and tasks using Hibernate relationship mapping.

The project was built to gain practical understanding of:
- how entities are connected
- how DTOs are used
- how services handle business logic
- how repositories communicate with the database
- how controllers expose APIs
- how Hibernate automatically manages relationships and foreign keys

This project helped in understanding industry-level backend development fundamentals and layered architecture using Spring Boot.
